**Original post:** [View on Discord](https://discord.com/channels/1325008017015701504/1469543578341605569)

This decoder takes two decimal digits and outputs a pulse at one of 100 slices corresponding to the code.
## Features
- Constant time decoding. 30gt latency.
- Slim profile. 4 blocks wide.
- Hopperspeed throughput.
- QC based logic with state auto-fix line.
