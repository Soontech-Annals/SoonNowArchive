This decoder takes two decimal digits and outputs a pulse at one of 100 slices corresponding to the code. The signal travels 10 blocks per second giving the device an O(n) time complexity.
## Features
- O(n) linear time decoding, 2 gt/block. Signal travels 10 blocks per second.
- Slim profile. 4 blocks wide.
- Hopperspeed throughput.
- QC based logic with state auto-fix line.
- Extremely lag-friendly, produces almost no lag.
