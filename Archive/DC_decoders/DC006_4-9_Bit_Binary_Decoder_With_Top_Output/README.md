# 4-9 Bit Binary Decoder With Top Output

**Tags:** *Functional*

**Original post:** [View on Discord](https://discord.com/channels/1325008017015701504/1469545656476631140)

This decoder takes 4 bits and outputs a pulse at one of 16 slices corresponding to the code. The device can be expanded to 9 bits of input. The output can be taken from the top of the device.
## Features
- Can be expanded to up to 9 bits of input.
- Outputs can be taken from the top. (Observe rail state for output)
- 11 gt latency and 8 gt throughput.
