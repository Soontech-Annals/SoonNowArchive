This quad display slice has 4 box displays with passive read ability. It has a fully hopperlocked layout with no visible pistons. The bottom displays has one buffer box with global first box placement. Box collection is not fully reliable. Top display will break if used aggressively by activating with adjacent slices. Two alternative layouts are available.
## Features
- Fully hopperlocked with local top display locking
- Waterstream inputs
- Passive read restock system with in-slice lists
- Global first box placement
## Considerations
- Box collection is not fully reliable
- Top display will break if used aggressively by activating with adjacent slices
