# Minewave Fixed Bulk Slice
<img alt="area_render_15_.png" src="images/area_render_15_.png?raw=1" height="300px">

**Authors:** *Basil*

**Tags:** *Functional*

**Original post:** [View on Discord](https://discord.com/channels/1325008017015701504/1469573119646892033)

A device that can be stacked to create a fixed external bulk storage system. The device interfaces with an decoder (not included) and a controller (not included) to automatically store and retrieve items with redstone control. Multi-slicing, the ability to set multiple slices to the same code, is supported. These multi-sliced slices need not be adjacent.
## Features
- 1.1 million item capacity
- Multi-slicing compatible
- Input overflow protection
- Fully hopperlocked
