This encoder uses multiple hoppercarts to allow for hopperspeed item to code encoding. The design shown is a 54 double-chest design which allows for up to 1000 item types with a decimal encoding scheme.
## Features
- Hopperspeed (9000 items/hr) encoding.
- Adjustable and expandable to work with any base.
## Considerations
- Uses 30 carts and floating rail. (designed before cart yeeting)
- Requires 4 items per item type per chest.
- Laggy at full speed
