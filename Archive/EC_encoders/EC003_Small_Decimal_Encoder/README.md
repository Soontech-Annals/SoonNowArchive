This small item to decimal encoder uses demibolts and a simple layered design. It can handle up to 1000 item types.
## Features
- Small and simple stacked design with demibolts.
- Encodes up to 1000 item types.
- Uses up to four carts for 1 code/56 gt throughput
## Considerations
- Designed before cart yeeting, stores carts at input
