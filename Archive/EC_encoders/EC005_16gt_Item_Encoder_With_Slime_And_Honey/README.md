This encoder has a 16gt throughput encoder core which only requires one item per type in a chest. It uses slime and honey in its wiring. Head not included.
## Features
- Half-hopperspeed (16gt) throughput
- Requires only 1 item per type per chest.
- No piston toggle states.
## Considerations
- Uses slime and honey in wiring
