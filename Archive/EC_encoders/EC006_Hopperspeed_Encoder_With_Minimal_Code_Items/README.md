This encoder uses multiple hoppercarts to allow for hopperspeed item to code encoding. Unlike other hopperspeed encoders, this design only requires 2 items per item type per chest, the minimum amount possible.
## Features
- Hopperspeed (9000 items/hr) encoding.
- Adjustable and expandable to work with any base.
- Requires only 2 items per item type per chest.
- Cart yeeter based cart recycler and storage.
