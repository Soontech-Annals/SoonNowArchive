The EC06 Item to Binary Encoder With Safety Features is a 10 bit encoder with up to 1024 codes. It has safety features to prevent errors, namely dry fire protection and detection. It uses bit grouping to reduce the amount of chests needed for encoding.
## Features
- 10 bit encoding with up to 1024 codes
- Easily accessible coding chests
- Safety features to prevent errors
- Dry fire protection and detection
