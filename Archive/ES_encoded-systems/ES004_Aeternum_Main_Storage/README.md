The Aeternum SMP storage system was designed as a middle ground between a standard storage system and an encoded one, aiming to be easy to build and use. It includes all the essentials that anyone might need from a storage system, with a strong focus on practicality and reliability.
## Features
- Regular Version 97.44% Hopper Locking - Essential Version 98.70% Hopper Locking
- 1.7 MSPT idle, 3 MSPT active (Ryzen 9 7950X3D with Lithium 1.21.4)
- 744 chest hall items, one chest per item
- 248 external bulk items, 0.95 million capacity per item
- 7 multi-item categories
- Unstackable sorter + encoder
- Set-based unloading array
- Some peripherals (furnace array, brewer, mass crafter, box dyer, restock display)
- System status panel
- Basic error checking, which can pause the input if there is a problem.
