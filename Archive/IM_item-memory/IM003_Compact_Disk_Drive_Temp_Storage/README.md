Stores boxes in a temporary storage system. The device is designed to be used in a dynamic sorting system to merge partial boxes.
## Features
- Stores 32 boxes in each double chest for high density
- 8 bit address, up to 256 boxes
- Built in isBox check for easy pair detection
- Safety feature checks if chest is reset properly
- Safety feature checks if placeholder items are missing
## Instructions
1. To use, set an 8 bit address and input a box.
2. The drive will iterate through the boxes in the right chest to find a box already stored in the system.
3. If a box is found, both boxes are outputted.
4. If no box is found, the input box is stored in the drive for later pairing.
5. The drive can store up to 256 boxes (one per address) in 8 double chests.
