# Tiny Request Manager
<img alt="pic.png" src="images/pic.png?raw=1" height="300px">

**Authors:** *Basil, heilzmaker, skyzy*

**Tags:** *Functional*

**Original post:** [View on Discord](https://discord.com/channels/1325008017015701504/1469767287568597013)

A device that can be used to queue requests in a system. It ensures that only one request is processed at a time regardless of random request timings. The device is infinitely expandable and can be paused.
## Features
- Random input proof
- Infinitely expandable
- Pausable
- Compact. 8x4x5 volume.
## Notes
Reset input is not rapid input proof.
