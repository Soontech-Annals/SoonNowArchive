# Mini Shulker Box Merger
<img alt="merger.png" src="images/merger.png?raw=1" height="300px">

**Authors:** *Kizu, skyzy*

**Tags:** *Functional*

**Original post:** [View on Discord](https://discord.com/channels/1325008017015701504/1481323593924214978)

A relatively simple device that takes shulker boxes in pairs and merges the boxes in each pair together. Based on the minimum hopper layout by Floppy ([MG002 Minimum Hopper Box Merger](https://discord.com/channels/1325008017015701504/1470486385814278155))
## Features
- Small; 6x8 slice, 4w frontend
- 26 hoppers, 0 locked (FHU)
- 8 slices for a total of 8x hopperspeed merging
- Status outputs for HasItems, IsActive and SlicesAvailable
