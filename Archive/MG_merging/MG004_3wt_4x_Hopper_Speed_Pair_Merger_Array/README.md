The slice logic is almost the same as a hopper speed merger, but uses hopper minecarts to achieve four-speed merging
## Features
- 4x hopper speed merging.
- Hopper minecarts auto-recover when idle, with no entities.
- Fully Hopper Locked.
- It has a built-in fill level comparer, and after merging, the output goes through a fill level sorter.
- Expandable.
## Notes
* A pair of partial boxes must be input simultaneously and at a period no less than 25gt.
* Each unit expansion requires adding 6gt to the box distribution period.
