The MX02 Jeopardy Theme Loop generates music to the tune of the [Jeopardy! Theme](https://www.youtube.com/watch?v=vWuQVpBeqLs) song.
## Features
- Repeatedly outputs the Jeopardy! theme song
- Music duration: 256 gt
